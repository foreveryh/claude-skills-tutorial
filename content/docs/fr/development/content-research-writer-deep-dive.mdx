## Analyse Approfondie : Workflow d'Écriture Collaborative

### Phase 1 : Configuration de l'Environnement

Avant de commencer, le skill recommande de créer un répertoire de projet dédié :

```bash
mkdir ~/writing/my-article-title
cd ~/writing/my-article-title
touch article-draft.md
```

Cette approche structurée garantit que tous les fichiers connexes (plans, recherche, brouillons, feedback) sont gérés centralement, facilitant le contrôle de version et la collaboration [2].

### Phase 2 : Création Collaborative de Plan

Le skill fournit un modèle de plan détaillé (lignes 89-139) pour aider à transformer les idées initiales en un cadre structuré :

```markdown
# Plan d'Article : [Titre]

## Accroche
- [Ligne d'ouverture/histoire/statistique]
- [Pourquoi le lecteur devrait s'intéresser]

## Introduction
- Contexte et fond
- Énoncé du problème
- Ce que cet article couvre

## Sections Principales

### Section 1 : [Titre]
- Point clé A
- Point clé B
- Exemple/preuve
- [Recherche nécessaire : sujet spécifique]

### Section 2 : [Titre]
- Point clé C
- Point clé D
- Données/citation nécessaire

## Conclusion
- Résumé des points principaux
- Appel à l'action
- Pensée finale

## Recherche à Faire
- [ ] Trouver des données sur [sujet]
- [ ] Obtenir des exemples de [concept]
- [ ] Source de citation pour [affirmation]
```

La clé est l'**optimisation itérative** — ne pas compléter le plan en une seule fois, mais ajuster répétitivement la structure, garantir un flux logique, identifier les lacunes de recherche et marquer les zones nécessitant une exploration approfondie [3].

### Phase 3 : Assistance de Recherche et Citation

C'est l'une des fonctionnalités les plus économiseuses de temps du skill. Lorsque les utilisateurs demandent une recherche sur un sujet, le skill :

1. Cherche des informations pertinentes
2. Trouve des sources crédibles
3. Extrait des faits clés, des citations et des données
4. Ajoute des citations dans le format demandé

**Exemple de Sortie** (lignes 149-171) :

```markdown
## Recherche : Impact de l'IA sur la Productivité

Principales Découvertes :

1. **Gains de Productivité** : Les études montrent 40% d'économies de temps
   pour les tâches de création de contenu [1]

2. **Taux d'Adoption** : 67% des travailleurs du savoir utilisent des outils IA
   hebdomadairement [2]

3. **Citation d'Expert** : "L'IA augmente plutôt qu'elle ne remplace la créativité
   humaine" - Dr. Jane Smith, MIT [3]

Citations :
[1] McKinsey Global Institute. (2024). "Le Potentiel Économique
    de l'IA Générative"
[2] Stack Overflow Developer Survey (2024)
[3] Smith, J. (2024). Entretien MIT Technology Review

Ajouté au plan sous la Section 2.
```

Cette sortie de recherche structurée peut être directement intégrée dans votre plan, économisant un temps considérable sur la recherche manuelle et le formatage [4].
