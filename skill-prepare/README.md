# Fumadocs 文章导入 Skill 准备文档

## 项目概述

创建一个 Claude Skill，用于自动化将外部文章导入到 Fumadocs 项目中，包括：
- 下载原文内容和图片
- 多语言翻译（4种语言）
- 文章分类和归档
- MDX 格式转换和验证
