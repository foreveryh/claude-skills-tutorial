# 需求详细说明

## 功能需求

### 核心功能
用户输入一个文章的 URL，Skill 自动完成以下任务：

1. **下载文章原文**（文字 + 图片）
2. **文章归类**（放到合适的 category 下）
3. **多语言翻译**（4种语言）
4. **格式转换**（转换为 Fumadocs MDX 格式）
5. **源文件归档**（保存原文和图片）
